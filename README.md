# beep-api

```
git clone https://github.com/implicit-invocation/beep-api.git
cd beep-api
npm i
npm run dev
```

Install mongodb: https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/  
Install postman: https://www.postman.com/downloads/
